TSV-Only Grid Copy Test

Reproduces a custom data grid whose copy handler writes only tab-separated text to text/plain — no text/html is placed on the clipboard. This is the trigger state the shared TsvDetector is built to handle: both Cmd+Shift+U and Cmd+Shift+O should detect it and populate the clipboard's text/html with the simulated simple HTML (a <table>).
